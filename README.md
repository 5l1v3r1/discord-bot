# Discord-Bot

A simple template for a telegram bot

# How it works

Using the discord.js module and an API-KEY of Discord builds a DiscordBot which serves music.

# Requirements

- NodeJS
- discord.js
- discord-youtube-api
- node-opus
- ytdl-core
